## 2. Deployment / Reproduction

### 2.1 Project materials

Clone this repository into your local drive, project structure would be like:

```
├───masks
├───Plots
│   └───Prediction
├───Processed
├───Source
│   ├───coe4
│   └───coe6
├───preproc.py
└───main.ipynb
```

The processing program is `main.ipynb` with description, other `.py` files are drafts. The `.ipynb` file name in Chinese is written by Yuchen Guo.

### 2.2 Environment setup

This project is based on Python 3.5 and managed by Conda.
